# Wedding Invitation Website

A simple, responsive static website for a wedding invitation.

## Project Structure

- `index.html`: Main page structure using Custom Elements.
- `css/`: Contains `style.css` for global styling and `components/` for component-specific styles.
- `js/`: 
    - `slider.js`: Logic for image slider.
    - `countdown.js`: Logic for the countdown timer.
    - `components/`: Custom Element definitions for `wedding-slider`, `wedding-countdown`, `wedding-cards`, and `wedding-footer`.
- `img/`: Folder for images.
- `components/`: Original HTML snippets for reference.

## Design

The design follows a warm, elegant aesthetic with:
- A handwritten title font (`Great Vibes`).
- A classic serif font for body text (`Playfair Display`).
- A soft gradient background.
- Rounded card components with soft shadows.

## How to use

Simply open `index.html` in any modern web browser to view the invitation.
